# Supersim

For comprehensive Supersim documentation, please see the [Supersim guide in the Interop section](/interop/tools/supersim).

Supersim is a local development environment that simulates multiple chains for testing interoperability features. It provides:

- Multi-chain local development environment
- Interop testing capabilities  
- Superchain ERC20 token testing
- Cross-chain message passing simulation
